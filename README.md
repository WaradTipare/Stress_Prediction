\# 🎯 Student Stress Prediction System



\## 📌 Overview



A machine learning-based web application that predicts student stress levels using lifestyle factors.



\## 🚀 Features



\* Stress prediction (Low / Medium / High)

\* Interactive gauge visualization

\* Personalized suggestions

\* Mental health support



\## 🛠️ Tech Stack



\* Python

\* Scikit-learn

\* Streamlit

\* Plotly



\## ▶️ Run the App



```bash

pip install -r requirements.txt

streamlit run app/app.py

```
