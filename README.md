# 🎯 Student Stress Prediction System

## 📌 Overview

A machine learning-based web application that predicts student stress levels using lifestyle factors like sleep, study hours, screen time, workload, and mood.

---

## 🚀 Features

* Stress level prediction (Low / Medium / High)
* Interactive gauge visualization
* Personalized suggestions
* Mental health support helpline

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Streamlit
* Plotly
* Pandas, NumPy

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
```

---

## 📊 How it Works

1. User inputs lifestyle data
2. Data is scaled and passed to ML model
3. Model predicts stress level
4. Custom stress score is used for visualization
5. Suggestions and support are provided
